Java SE 7 Update 21 will require code signing for all code run in the browser, not just for code that required elevated permission. On one hand it's sad that this means no more unsigned apps for simple interactivity (I had some nice Java based animated buttons back in the day), on the other hand modern browser features are probably a better approach.
