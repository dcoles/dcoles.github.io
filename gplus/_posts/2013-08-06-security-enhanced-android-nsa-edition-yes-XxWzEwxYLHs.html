"Security-Enhanced Android: NSA Edition"
Yes, the NSA has contributed code to Open Source projects (including Android), no it's not some sort of secret NSA eavesdropping code (it's a whole bunch of optional security enhancements that you don't even have to use to make it harder to exploit). And the best part is if you don't believe me, you or anyone else can go look at the code and check what it does for yourself.
